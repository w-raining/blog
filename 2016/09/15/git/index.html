git

git的使用

git的命令行

git checkout -b user   创建并切换user分支
git branch user        创建分支
git checkout user      切换分支
------------------------------------------
git last               查看最后一条记录
git log                用于查看历史纪录
------------------------------------------
解决冲突
git status             查看状态 git会告诉我们冲突的文件
git checkout -b aa     创建一个新的分支 并切换

git add aa.html        提交
git commit -m '1.0'
git push   
git branch -d aa
git log                查看分支的合并情况

git br -d aa           删除aa分支  
-------------------------------------------
git pull               推送代码
